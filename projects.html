04 : projects

With so much to learn in tech, the best way to learn is by doing with projects. Here are some of the projects i've worked on and the technologies used.

001/003

Self Driving RC Car

I built a RC car using a raspberry pi, arduino, L293n motor controller, and some motors. I'm currently working on adding self-driving capabilities using CNNs for lane detection.

Tech Stack: ROS2, Python, C++, TensorFlow/Keras

002/003

Image Search Engine

Built a backend application with Python and OpenCV to search for similar looking images. A SQL database stores locations of images and the feature vectors describing the images as histograms of colors.

Tech Stack: Python, MySQL, OpenCV

003/003

Culture Hub Web App

Worked on a team of interns to build a web app for a company culture hub for TQL Hackathon. We placed 5 out of 16 against teams of fulltime developers.

Tech Stack: Angular, C#/.NET, CosmosDB (NoSQL)
